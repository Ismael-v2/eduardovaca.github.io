ESP32 Climate Display

This project displays temperature and humidity on an OLED screen using an ESP32 and DHT11 sensor.

  • Microcontroller: ESP32
  • Sensor: DHT11 (Temperature & Humidity)
  • Display: 0.96" I2C OLED
  • Code Language: Arduino (C++)

Project Features:

  • Real-time climate monitoring
  • OLED display output via I2C
  • ESP32 web server support (optional)
